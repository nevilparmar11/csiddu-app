# csiddu-app
- App is built using flutter and Firebase.

## Features

## Normal User
- SignUp , Signin with Google
- Register for the event
- View the list of upcoming and past events
- Profile page
- Marks Predictor

## Admin
- Add new event 
- Manage event registrations
- Update event details
- Manage users
